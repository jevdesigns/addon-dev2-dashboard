## Changelog

All notable changes to this add-on will be documented in this file.

### [1.0.0] - 2025-12-06

#### Added
- Initial release of Dev2 React Dashboard add-on
- Ingress support for seamless Home Assistant integration
- Real-time entity state synchronization
- HomeKit-inspired UI with Tailwind CSS
- Local storage using IndexedDB for entity history
- Responsive design for all device sizes
- Theme support (auto, light, dark)
- Configurable logging levels

#### Features
- Automatic authentication via Home Assistant ingress
- WebSocket connection for real-time updates
- Service call integration
- Entity history tracking
- Modern, polished interface
